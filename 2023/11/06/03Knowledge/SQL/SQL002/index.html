统计每一关开始了多少次

select
count(事件名)
FROM test.例子表
where 事件名 = 'level_start'
group by 关卡号

统计每一关开始了多少次的另一种方法

select
SUM(case 事件名 when 'level_start' then 1 else 0 end) as 开始次数
FROM test.例子表
where 事件名 IN ('level_start', 'level_succ')
group by 关卡号

我疑惑的是，下面这种写法，多写一行就不行了：

select
SUM(case 事件名 when 'level_start' then 1 else 0 end) as 开始次数
SUM(case 事件名 when 'level_succ' then 1 else 0 end) as 成功次数
FROM test.例子表
where 事件名 IN ('level_start', 'level_succ')
group by 关卡号
