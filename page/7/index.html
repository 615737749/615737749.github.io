你的名字

一个叫“泷”的男高中生，住在东京，擅长素描，周末在饭店打工，喜欢饭店的一个女生，她叫寺岛，年龄比泷稍长几岁，是服务生的领班。饭店里的其他男服务生也喜欢寺岛，不过都没有表白，都跟泷一样，属于单相思，也许泷喜欢得更多一点吧。

一个叫“三叶”的女生，住在一个叫“系守镇”的地方，过着普通的乡村生活，村子围湖而建，湖是很多年前陨石砸出一个大坑后慢慢形成的。村里人信奉神明，三叶家族照看着村里神社，三叶的父亲因妻子的过世而自疚，离开了神社，后来当上了镇长。三叶与父亲间有些许矛盾。三叶觉得村里的生活很苦闷，一直憧憬着东京的生活。

两个地方，没有交集的两个人在神明的作用下，互换了身体，具体来说就是一觉醒来，就互换了身体，发生这种互换是没有规律的，一周大约会互换两三次。就这样，进入了彼此的生活，泷扮演着三叶在学校里做了些一般女生做不到也不会做的事，比如坐姿豪放，打篮球等，引得了其他男生的爱慕。三叶扮演着泷在饭店做了些一般男生做不到的事，比如缝补衣服，引得了寺岛的欣赏，并逐渐追求到了跟寺岛约会的机会。两人从一开始的不适应彼此身体到约法三章，到为对方付出。但好景不长，泷知道即将到来的流星会击中系守镇的某个地方，自然三叶也就知道了。所以两人都在为了系守镇村民不受陨石伤害而努力，并最终挽救了村民的生命。慢慢的，两人喜欢上了彼此。

但泷所见到的三叶是三年前的三叶，虽然彼此经历了那么多，但时空的交错，终使两人不能相见。陨石灾难之后，神明剪断了两人的连线，再也没有发生身体交换了，而且忘了彼此的名字，但两人共同经历过很多事情，有很多共同的回忆，相互喜欢，都不想忘记对方。所以，在陨石灾难八年后，泷和三叶再次在东京相见，陌生而又熟悉，找到彼此心里所一直寻找的熟悉感时，两人都已泪目。

剧情分：穿越、身体交换、因预知灾难而努力拯救无辜。这些点比较俗套啦。
搞笑分：身体交换的笑点，摸胸。地域交换的笑点，乡下人去城里通常该有的笑点。傲娇小萝莉属性所带来的的笑点：四叶的日常卖萌和神吐槽。
槽点：1.画蛇添足的大圆满结局，是为了迎合市场而做的妥协。2.藤井树VS名字，山难VS陨石，渡边博子去找藤井树VS泷去找三叶。3.泷和三叶在一段阶梯上相认，好像次郎和机器人女友在阶梯上的故事，有毒。。。。

谥号、庙号、年号

发表于 2021-04-13 分类于 05历史

谥号：有地位的人（皇帝、诸侯、文人等）死后别人给取的，评价生平。褒贬都有。拓跋宏的谥号是孝文皇帝，常称他为北魏孝文帝。
庙号：皇帝死后，太庙供奉的称号。唐朝到元朝的皇帝多称庙号。如李世民的庙号是太宗，常称他为唐太宗。
年号：纪年用的。明清两朝皇帝多称年号。如爱新觉罗·玄烨的年号是康熙，常称他为康熙帝。
尊号：活着的时候别人给起的尊称。

最早的是年号是汉武帝的“建元”，最后的年号是溥仪的“宣统”。
年号最多的是武则天，用了17个。
年号用的最久的是康熙，用了61年。
年号最短的是金末帝的“盛昌”，用了一天。

使用Hexo建立博客

发表于 2021-04-13 分类于 04博客

注：写给自己看的，所以写得比较简单。

远程准备工作

注册 GitHub（或者 Gitee）账号，建立远程仓库（仓库名同用户名）。

生成 SSH 公钥，添加公钥，测试连接。

本地准备工作-配置环境

  1. 安装 Git，检查 Git 环境 git --version。

  2. 安装 Node.js，检查 Node.js 环境 node -v。

    检查 npm 版本 npm -v，更新 npm 版本 npm install npm@latest -g。

  3. 安装 Hexo npm install -g hexo-cli。

本地准备工作-初始化博客目录

  1. 新建文件夹，执行 hexo init（Mac 系统下是 npx hexo init）来初始化该目录 。

本地准备工作-安装插件

  1. 安装主题插件NexT npm install hexo-theme-next。

    其他我比较喜欢的主题有：Oranges、Chic、3-hexo、Cactus-white、yinwang。

  2. 安装部署插件 npm install hexo-deployer-git。

  3. 安装盘古插件 npm install pangu。

  4. 安装加密插件 npm install hexo-blog-encrypt。

  5. 安装懒加载图片插件 npm install hexo-lazyload-image。

注：npm 安装模块时，跟了 -g 的就是全局安装，没跟就是本地安装。跟了 --save 的会被写入到生产环境里。

注：可以通过 npm list（项目级别） 和 npm list -g（全局） 来查看模块到安装信息。

本地准备工作-同步远程文章

  1. 删除 source 文件夹和 _config.yml 文件和 .gitignore 文件。

    拷贝 source 文件夹、.gitignore 文件、_config.next.yml 文件、_config.yml 。

    初始化成Git仓库：git init。

    关联远程仓库：git remote add origin 仓库地址。

    拉取仓库内容：git pull origin master。

    设置默认分支：git branch --set-upstream-to=origin/master master。

如何让终端使用VPN

发表于 2021-04-10 分类于 02工具

如何让终端使用代理

  1. MacOS 在终端里输入：

    export all_proxy=socks5://127.0.0.1:1081

  2. Windows 在命令提示符里输入：

    set all_proxy=socks5://127.0.0.1:1081

如何测试终端是否有使用代理

输入 curl www.google.com 。

如果返回类似 curl: (7) Failed to connect to www.google.com port 80: Timed out 说明没有走代理。

如果返回一个以 <!doctype html> 开头的 html 源码的话，说明有走代理。

说明

  1. 上边例子的 Socks5 端口是 1081，不是固定的，根据你所使用的代理软件来设置。

  2. all_proxy 就表示 http 和 https 的合集，你也可以分开来指定。
